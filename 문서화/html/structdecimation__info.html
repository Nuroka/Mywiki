상세한 설명

The weight grid information for a single decimation pattern.

ASTC can store one weight per texel, but is also capable of storing lower resolution weight grids that are interpolated during decompression to assign a with to a texel. Storing fewer weights can free up a substantial amount of bits that we can then spend on more useful things, such as more accurate endpoints and weights, or additional partitions.

This data structure is used to store information about a single weight grid decimation pattern, for a single block size.
