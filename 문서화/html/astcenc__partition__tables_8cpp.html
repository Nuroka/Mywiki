static void generate_canonical_partitioning ( unsigned int texel_count,
const uint8_t * partition_of_texel,
uint64_t bit_pattern[BIT_PATTERN_WORDS] )
static

Generate a canonical representation of a partition pattern.

The returned value stores two bits per texel, for up to 6x6x6 texels, where the two bits store the remapped texel index. Remapping ensures that we only match on the partition pattern, independent of the partition order generated by the hash.

매개변수
texel_countThe number of texels in the block.
partition_of_texelThe partition assignments, in hash order.
[out]bit_patternThe output bit pattern representation.
