Functions for the library entrypoint.

Record of the quality tuning parameter values.

See the astcenc_config structure for detailed parameter documentation.

Note that the mse_overshoot entries are scaling factors relative to the base MSE to hit db_limit. A 20% overshoot is harder to hit for a higher base db_limit, so we may actually use lower ratios for the more through search presets because the underlying db_limit is so much higher.
