\doxysubsection{상세한 설명}
Weight quantization transfer table. 

ASTC can store texel weights at many quantization levels, so for performance we store essential information about each level as a precomputed data structure. Unquantized weights are integers or floats in the range \mbox{[}0, 64\mbox{]}.

This structure provides a table, used to estimate the closest quantized weight for a given floating-\/point weight. For each quantized weight, the corresponding unquantized values. For each quantized weight, a previous-\/value and a next-\/value.
