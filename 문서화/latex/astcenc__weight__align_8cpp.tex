Functions for angular-\/sum algorithm for weight alignment. 

This algorithm works as follows\+:
\begin{DoxyItemize}
\item we compute a complex number P as (cos s\texorpdfstring{$\ast$}{*}i, sin s\texorpdfstring{$\ast$}{*}i) for each weight, where i is the input value and s is a scaling factor based on the spacing between the weights.
\item we then add together complex numbers for all the weights.
\item we then compute the length and angle of the resulting sum.
\end{DoxyItemize}

This should produce the following results\+:
\begin{DoxyItemize}
\item perfect alignment results in a vector whose length is equal to the sum of lengths of all inputs
\item even distribution results in a vector of length 0.
\item all samples identical results in perfect alignment for every scaling.
\end{DoxyItemize}

For each scaling factor within a given set, we compute an alignment factor from 0 to 1. This should then result in some scalings standing out as having particularly good alignment factors; we can use this to produce a set of candidate scale/shift values for various quantization levels; we should then actually try them and see what happens.
